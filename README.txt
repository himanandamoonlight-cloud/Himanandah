Himānandaḥ — The Moonlight of the Soul
Static dashboard demo

Files:
- index.html : homepage
- styles.css : simple styles with light/dark toggle

How to preview locally:
1) Open index.html in a browser.
2) Or run a tiny static server:
   python3 -m http.server 8000
   and open http://localhost:8000

To deploy to Vercel:
- Create a new project on Vercel and import this repository or upload the folder.
- Set framework to 'Other' / Static.
